ArticTools is a collection of free tools for use in speech therapy for articulation.

Find Pictograms

Search for color pictograms using keywords. This lets you avoid using verbal modeling and focus on spontaneous production.

Generate target word lists using different phonological approaches by using minimal pairs or a maximal contrast approach.

Create Worksheets

Combine pictograms with target words to create worksheets for therapy sessions or home practice.
